- **Frontend**: React 18 + Vite

- **Realtime**: Socket.IO
- **Database**: MongoDB
- **Authentication**: JWT
- **Validation**: Joi
- **Optional**: Redis (caching, session management)

## рҹҡҖ Getting Started

### Prerequisites
- Node.js >= 16
- MongoDB >= 4.4
- npm or yarn

### Installation

1. **Clone & Install Backend**
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your MongoDB URI and JWT secret
npm run dev
```

2. **Install Frontend**
```bash
cd frontend
npm install
npm run dev
```

3. **Access the App**
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
